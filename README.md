## ğŸš€ Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources (Three.js, Typed.js, Font Awesome)

### Local Development

1. **Clone the repository**

   ```bash
   git clone https://github.com/iamvnetxAI/iamvnetxAI.github.io.git
   cd iamvnetxAI.github.io
   ```

2. **Open locally**

   ```bash
   # Option 1: Direct file opening
   open index.html

   # Option 2: Using Python HTTP server
   python -m http.server 8000
   # Then visit http://localhost:8000

   # Option 3: Using Node.js live-server
   npx live-server
   ```

3. **View in browser**
   - Navigate to the opened file or localhost address
   - The website should load with all animations and effects

### Colors & Themes

The website uses CSS custom properties for easy theming:

```css
:root {
  --accent-blue: #00d4ff;
  --accent-purple: #8b5cf6;
  --accent-pink: #f472b6;
```

### Animations

Animations can be customized in the CSS file:

- Adjust timing with `animation-duration`
- Modify easing with `animation-timing-function`
- Change delay with `animation-delay`
